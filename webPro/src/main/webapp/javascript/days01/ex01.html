	[css 변수 선언 + 사용]
	- 지역변수, 전역변수 선언 가능

	- javascript, 미디어 쿼리로 변수 변경 가능
body { background-color: #1e90ff; } h2 { border-bottom: 2px solid #1e90ff; } .container { color: #1e90ff; background-color: #ffffff; padding: 15px; } button { background-color: #ffffff; color: #1e90ff; border: 1px solid #1e90ff; padding: 5px; } - 코드 가독성 좋아짐 -> 유지보수 용이 전역변수 선언 :root { --변수명:값; --blue:#1e90ff; --white:#ffffff; } body { background-color: var(--blue); } h2 { border-bottom: 2px solid var(--blue); } .container { color: var(--blue); background-color: var(--white); padding: 15px; } button { 전역변수 --blue 재정의 -> 지역변수 --blue: #0000ff; --button-blue: #0000ff; -> 새로운 지역변수 선언 background-color: var(--white); color: var(--blue); border: 1px solid var(--blue); padding: 5px; }
