[display 속성]
	1. 레이아웃(배치)을 제어하기 위한 가장 중요한 css 속성
	2. 기본값: block, inline
	   block - div, p, header, hr, hn 등
	   특징) 항상 새로운 라인에서 시작
	   		부모 너비의 100%가 잡힘
	   		
	   inline - span, a, img 등
	   특징) 새로운 라인에서 시작하지 않음
	   		너비는 필요한 내용(content)만큼만 잡힘
