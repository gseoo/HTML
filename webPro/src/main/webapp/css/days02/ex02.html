html 여백(margin)

	여백(margin)
	테두리(border)
	패딩(padding)
	
	margin: 10px -> 약식
	
	margin-top: 속성값;
	margin-right: 속성값;
	margin-bottom: 속성값;
	margin-left: 속성값
	
	속성값
	- auto : 브라우저가 여백을 자동으로 설정
	- px, pt, cm 등 길이 단위
	- 부모 요소를 기준으로 % 단위
	- inherit() : 상속 - 부모 여백을 자식한테 그대로 설정
	
	margin-top: 25px;
	margin-bottom: 75px;
	이라도 100px이 아님 75px이 됨 (여백의 붕괴)
