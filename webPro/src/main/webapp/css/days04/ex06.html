[css !important 규칙]

		!important
		- CSS 규칙, 일반적인 것보다 더 중요성을 나타냄
		- 요소의 특정 속성에 대해 이전 모든 스타일 규치을 재정의한다.

		[css 특이성]
		1. 인라인식 스타일 시트 적용 : 1000점
		2. * : 0점
			태그명, ::의사요소 : 1점
			.클래스명, :의사클래스, 속성 선택자 : 10점
			#아이디명 : 100점
		예) #navlist li#home -> 100 + 1 + 100 = 201점
