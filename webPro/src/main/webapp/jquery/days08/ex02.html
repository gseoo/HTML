1. jquery? 1) jquery는 빠르고 작으며 기능이 풍부한 javascript 라이브러리이다. 2) 다양한 브라우저에서 작동하는 사용하기 쉬운 API를 통해 HTML 문서 탐색 및 조작, 이벤트 처리, 애니메이션, Ajax 등을 훨씬 간단하게 만든다. 3) 다용성과 확장성을 결합한 jQuery는 수백만 명의 사람들이 JavaScript를 작성하는 방식을 변화시켰다. 2. jquery를 사용하는 방법? 1) jquery 파일의 압축 및 비압축 복사본 - 비압축 : 개발 or 디버깅 - 압축 : 대역폭을 절약하고 프로덕션 성능을 향상시킴 2) CDN은 전 세계에 분산된 서버에서 jQuery를 호스팅하여 성능상의 이점을 제공 3. jquery 사용하는 구문(형식) $(selector).action()
